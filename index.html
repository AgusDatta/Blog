Proyectos

SpaceX-Launches

Este proyecto es una página web dedicada a los lanzamientos de SpaceX, diseñada utilizando Astro y Tailwind CSS. La web se conecta a la API de SpaceX para extraer información detallada sobre cada lanzamiento. Los usuarios pueden ver información sobre el número de lanzamiento, si fue exitoso o fallido, y una descripción detallada de cada evento. Este proyecto fue desarrollado como una incursión en las tecnologías de Astro y Tailwind.
