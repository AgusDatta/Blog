Proyectos

SpaceX-Launches

Este proyecto es una página web dedicada a los lanzamientos de SpaceX.
La web se conecta a la API de SpaceX para extraer información detallada sobre cada lanzamiento.
Los usuarios pueden ver información sobre el número del lanzamiento, si fue exitoso o fallido, y una descripción detallada de cada evento.
Este proyecto fue desarrollado como una incursión en las tecnologías de Astro y Tailwind, guiado por "Midudev".
