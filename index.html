SpaceX-Launches

Este proyecto es una página web dedicada a los lanzamientos de SpaceX. La web se conecta a la API de SpaceX para extraer información detallada sobre cada lanzamiento. Los usuarios pueden ver información sobre el número del lanzamiento, si fue exitoso o fallido, y una descripción detallada de cada evento. Este proyecto fue desarrollado como una incursión en las tecnologías de Astro y Tailwind.

Fake-Shop

Este proyecto es una página web que simula un e-commerce moderno, permitiendo a los usuarios explorar productos, ver detalles en un modal dinámico y gestionarlos en un carrito de compras con persistencia mediante localStorage. Desarrollada con HTML5, CSS3 y JavaScript, integra Bootstrap 5 para diseño responsivo y SweetAlert2 para notificaciones interactivas. Los datos de los productos se obtienen dinámicamente desde la FakeStore API, ofreciendo una experiencia fluida y funcional con un diseño atractivo y escalable.
